### âœ… **Using ANSI Escape Codes (Basic)**

This method works in most Unix-like terminals (Linux/macOS, and modern Windows terminals):

| Style  | ANSI Code  |
| ------ | ---------- |
| Bold   | `\033[1m`  |
| Italic | `\033[3m`  |
| Reset  | `\033[0m`  |
| Red    | `\033[31m` |
| Green  | `\033[32m` |
| Yellow | `\033[33m` |
| Blue   | `\033[34m` |
| ...    | ...        |

#### Example:

```python
# Bold
print("\033[1mThis is bold text\033[0m")

# Italic
print("\033[3mThis is italic text\033[0m")

# Red and Bold
print("\033[1;31mThis is bold red text\033[0m")
```

ğŸ’¡ Always end with `\033[0m` to reset formatting.

### âœ… **Using `colorama` Library (Cross-platform, safer for Windows)**

Install with:

```bash
pip install colorama
```

Then:

```python
from colorama import init, Fore, Style

init()

print(Fore.RED + "This is red text" + Style.RESET_ALL)
print(Style.BRIGHT + "This is bold text" + Style.RESET_ALL)
```

> Note: `colorama` doesn't support italics out of the box (most terminals don't natively support it via Windows CMD/PowerShell).

### âœ… **Using `rich` Library (Best for Stylish Output)**

Install with:

```bash
pip install rich
```

Then:

```python
from rich import print

print("[bold]This is bold[/bold]")
print("[italic]This is italic[/italic]")
print("[bold red]This is bold and red[/bold red]")
print("[green]This is green[/green]")
```
